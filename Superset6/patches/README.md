# =============================================================================
# Superset 6.0.0 i18n Fix Patch - README
# =============================================================================
#
# このパッチは Superset 6.0.0 のフロントエンド翻訳の問題を修正します。
#
# 問題の詳細:
#   - https://github.com/apache/superset/issues/34751
#   - https://github.com/apache/superset/issues/35569
#
# 問題の原因:
#   Superset 6.0.0 では、プラグインや packages フォルダ内のコンポーネントが
#   翻訳関数 t('') を正しくインポートしていないため、チャート設定パネルや
#   ツールチップなどの一部の UI 要素が翻訳されません。
#
# このパッチの内容:
#   1. i18n-fix.sh      - 翻訳を適用するシェルスクリプト
#   2. i18n-fix.js      - Node.js版のパッチスクリプト（参考用）
#   3. templates/       - カスタムテンプレート（ランタイム修正を挿入）
#
# =============================================================================
# 公式修正がリリースされたら
# =============================================================================
#
# 以下の手順でこのパッチを削除してください:
#
# 1. Dockerfile から以下の行を削除:
#    - COPY patches/ /app/patches/
#    - RUN chmod +x /app/patches/i18n-fix.sh && /app/patches/i18n-fix.sh
#    - COPY patches/templates/ /app/superset/templates/
#
# 2. superset_config.py から以下の行を削除（もし追加していれば）:
#    - CUSTOM_TEMPLATE_DIR 関連の設定
#
# 3. patches/ フォルダ全体を削除
#
# =============================================================================
